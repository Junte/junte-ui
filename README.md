## For development:
On windows please install cygwin.

```
npm i
npm run iconfont
npm run encapsulation
npm run build:lib
npm start
ng serve
```

## For publication:
```
npm run package
npm login
npm publish ./dist/junte-ui/junte-ui-0.0.251.tgz
```
