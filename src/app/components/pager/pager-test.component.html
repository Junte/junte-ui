Pager

Requirements

  1. The page button should respond to mouse hover, click
  2. The active page button should be highlighted
  3. The pager must have next/prev buttons and first/last buttons
  4. Overflow must be considered in the pager
  5. The pager must have a page size selection
  6. The page button should change the display when it is disabled
