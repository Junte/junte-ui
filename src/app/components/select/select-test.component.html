Select

Requirements

  1. The select should show the loading
  2. The select and option list item should respond to mouse hover, click
  3. The select must have different sizes: small, normal, large
  4. The select should change the display when it is disabled
  5. The select should be of two types: single and multiple
  6. Selected items should be highlighted in option list
  7. The select must be able to enter a query
  8. The option list should be closed by clicking on free space or on the list item

Input property

Ajax query
