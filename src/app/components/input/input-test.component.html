Input

Requirements

  1. The input must have different sizes: small, normal, large
  2. The input should respond to mouse hover, focus
  3. The input should change the display when it is disabled
  4. The input must have a design in accordance with a given color scheme
  5. The button should be of two types: password and text
  6. The input must have a placeholder
