Skeleton

Requirements

  1. The avatar in skeleton must have different sizes: small, normal, large
  2. Skeleton can have avatar, title and paragraphs
  3. In the paragraph may change the count of rows
  4. Skeleton may have animation
 
<jnt-skeleton [title]="true" [avatar]="true" [rows]="5" [size]="ui.sizes.large"></jnt-skeleton>

API

jnt-skeleton

Property Description Type Default
[title] - true | false true
[avatar] - true | false false
[rows] - number -
