Progress bar

A view that depicts the progress of a task over time.
The Progress Bar class provides properties for managing the style of the progress bar and for getting and setting values that are pinned to the progress of a task.

Requirements

  1. Progress bar should display the percent complete
  2. If the percentage exceeds 100, the progress bar should change the background color
