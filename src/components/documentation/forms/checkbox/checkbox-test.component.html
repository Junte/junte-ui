Checkbox

Used to select a value from several options

Requirements

  1. The checkbox must have different sizes: tiny, small, normal, large
  2. The checkbox should change the display when it is disabled and disabled & checked
  3. The checkbox should respond to mouse hover, click
  4. Checkbox should work with reactive forms
